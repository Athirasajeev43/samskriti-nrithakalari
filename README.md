# Samskriti Nrithakalari

A Django-based web application for managing a dance academy.

## Features
- User Registration & Login (Email-based authentication)
- Course Selection (Bharatanatyam, Mohiniyattam, Kuchipudi, Folk Dance)
- Timetable Scheduling
- Admission Fee Payment System
- Instructor Details
- Gallery Management

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite

## Project Structure
- core/ → Main app logic
- templates/ → HTML files
- static/ → CSS & assets
- media/ → Uploaded images
